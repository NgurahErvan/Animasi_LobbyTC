## How To Run The Website

1. Clone Repository

```
git clone git@github.com:NgurahErvan/Animasi_LobbyTC.git
```

2. Install needed packages On Folder Terminal

```
npm install
```

3. Run the website On Folder Terminal

```
node app.js
```

4. Go To Browser And Search For

```
localhost:3000
```
